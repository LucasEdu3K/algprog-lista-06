# 📚 Exercícios de Vetores — Java

Repositório com exercícios acadêmicos de programação em Java, com foco em **vetores (arrays)**, estruturas de repetição e lógica de programação.

---

## 🎯 Objetivo

Praticar os conceitos fundamentais de Java, especialmente:
- Declaração e manipulação de vetores
- Estruturas de repetição (`for`)
- Estruturas condicionais (`if/else`)
- Entrada e saída de dados com `Scanner`
- Operações matemáticas com vetores

---

## 📂 Exercícios

| # | Arquivo | Descrição |
|---|---------|-----------|
| 01 | `Vetor5int.java` | Lê um vetor de 5 números inteiros e os exibe |
| 02 | `Vetor10inverso.java` | Lê um vetor de 10 números reais e exibe na ordem inversa |
| 03 | `MediaNotas.java` | Lê 4 notas, exibe e calcula a média |
| 04 | `LeitorConsoante.java` | Lê 10 caracteres e exibe as consoantes encontradas |
| 05 | `VetorImparPar.java` | Lê 20 inteiros e separa em vetores de pares e ímpares |
| 06 | `MediaAlunos.java` | Lê 4 notas de 10 alunos, calcula médias e conta aprovados |
| 07 | `VetorSomaMult.java` | Lê 5 inteiros e exibe a soma e a multiplicação |

---

## 🛠️ Tecnologias

- **Linguagem:** Java
- **Versão:** JDK 8+
- **IDE:** IntelliJ IDEA / Eclipse / VS Code

---

## ▶️ Como executar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Compile o arquivo desejado:
```bash
javac MediaAlunos.java
```

3. Execute:
```bash
java MediaAlunos
```

---

## 📖 Conceitos abordados

- **Vetores:** declaração, preenchimento e exibição
- **Loop `for`:** simples e aninhado (loop dentro de loop)
- **Condicionais `if/else`:** verificação de par/ímpar, vogal/consoante, aprovação
- **Operador módulo `%`:** verificar paridade
- **Formatação de saída:** `printf` com `%.2f`
- **Tipos de dados:** `int`, `double`, `char`
